# 📊 Data Science Internship Tasks

This repository contains solutions to selected advanced data science tasks completed as part of a Data Science Internship. The tasks focus on real-world applications including classification, clustering, and business analytics.

---

## 🚀 Tasks Overview

### 🔹 Task 1: Term Deposit Subscription Prediction
- **Objective:** Predict whether a customer will subscribe to a term deposit.
- **Approach:**
  - Data preprocessing and feature encoding
  - Logistic Regression model training
- **Evaluation Metrics:**
  - Confusion Matrix
  - ROC Curve
- **Outcome:** Successfully built a predictive model to support marketing decisions.

---

### 🔹 Task 2: Customer Segmentation
- **Objective:** Segment customers based on income and spending behavior.
- **Approach:**
  - Exploratory Data Analysis (EDA)
  - K-Means Clustering
- **Output:**
  - Visual clusters representing different customer groups
- **Outcome:** Identified customer segments for targeted marketing strategies.

---

### 🔹 Task 5: Sales Data Analysis Dashboard
- **Objective:** Analyze sales and profit performance.
- **Approach:**
  - Data cleaning and aggregation
  - Visualization using bar charts
- **Key Insights:**
  - Total Sales
  - Total Profit
  - Category-wise sales performance
